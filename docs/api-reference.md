# API Reference ÔÇö Movie Service

Base URL: `http://localhost:8083`

Write endpoints (`POST`, `PUT`, `DELETE`) require an admin JWT:
```
Authorization: Bearer <access_token>
```

Read endpoints (`GET`) are publicly accessible ÔÇö no token needed.

---

## Movies ÔÇö `/api/movies`

### POST /api/movies

Create a new movie.

**Admin** ┬À Returns `201 Created`

**Request body**
```json
{
  "title": "Inception",
  "genres": ["Sci-Fi", "Thriller"],
  "releaseYear": 2010,
  "description": "A thief who steals corporate secrets through dream-sharing technology.",
  "posterUrl": "https://example.com/inception.jpg"
}
```

**Validation rules**

| Field | Rules |
|-------|-------|
| `title` | Required, max 255 characters |
| `genres` | Optional list of strings |
| `releaseYear` | Optional, min 1990, max current year + 10 |
| `description` | Optional, max 2000 characters |
| `posterUrl` | Optional |

**Response**
```json
{
  "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "title": "Inception",
  "genres": ["Sci-Fi", "Thriller"],
  "releaseYear": 2010,
  "description": "A thief who steals corporate secrets through dream-sharing technology.",
  "posterUrl": "https://example.com/inception.jpg",
  "averageRating": null,
  "createdAt": "2025-01-01T12:00:00"
}
```

**Errors**

| Status | Reason |
|--------|--------|
| 400 | Validation failure (blank title, out-of-range year) |
| 401 | Missing or invalid JWT |
| 403 | Authenticated but not `ROLE_ADMIN` |

### GET /api/movies

List all movies with pagination and sorting.

**Public** ┬À Returns `200 OK`

**Query parameters**

| Parameter | Default | Description |
|-----------|---------|-------------|
| `page` | `0` | Zero-based page number |
| `size` | `20` | Page size |
| `sort` | `title` | Field to sort by |
| `direction` | `asc` | `asc` or `desc` |

**Response**
```json
{
  "content": [
    {
      "id": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
      "title": "Inception",
      "genres": ["Sci-Fi", "Thriller"],
      "releaseYear": 2010,
      "description": "...",
      "posterUrl": "https://example.com/inception.jpg",
      "averageRating": 8.8,
      "createdAt": "2025-01-01T12:00:00"
    }
  ],
  "page": 0,
  "size": 20,
  "totalElements": 1,
  "totalPages": 1
}
```

### GET /api/movies/search

Search and filter movies.

**Public** ┬À Returns `200 OK`

**Query parameters**

| Parameter | Required | Description |
|-----------|----------|-------------|
| `title` | No | Case-insensitive partial match on title |
| `genre` | No | Exact match on a genre in the genres list |
| `releaseYearFrom` | No | Minimum release year (inclusive) |
| `releaseYearTo` | No | Maximum release year (inclusive) |
| `page` | No (default `0`) | Zero-based page number |
| `size` | No (default `20`) | Page size |

All parameters are optional and combinable. Blank strings are treated as omitted.

**Example**
```
GET /api/movies/search?genre=Sci-Fi&releaseYearFrom=2000&releaseYearTo=2015
```

**Response** ÔÇö same shape as `GET /api/movies`.

---

### GET /api/movies/{id}

Get a single movie by ID.

**Public** ┬À Returns `200 OK`

**Response** ÔÇö single `MovieResponse` object (same shape as the item in the list above).

**Errors**

| Status | Reason |
|--------|--------|
| 400 | Movie not found with the given ID |

---

### PUT /api/movies/{id}

Update an existing movie.

**Admin** ┬À Returns `200 OK`

**Request body** ÔÇö same shape as `POST /api/movies`. All fields in the request replace the existing values.

**Response** ÔÇö updated `MovieResponse`.

**Errors**

| Status | Reason |
|--------|--------|
| 400 | Validation failure |
| 400 | Movie not found |
| 401 | Missing or invalid JWT |
| 403 | Authenticated but not `ROLE_ADMIN` |

---

### DELETE /api/movies/{id}

Delete a movie.

**Admin** ┬À Returns `204 No Content`

**Errors**

| Status | Reason |
|--------|--------|
| 400 | Movie not found |
| 401 | Missing or invalid JWT |
| 403 | Authenticated but not `ROLE_ADMIN` |

## Data Model

### Movie (Neo4j Node)

| Property | Type | Notes |
|----------|------|-------|
| `id` | String (UUID) | Generated by `UUIDStringGenerator` |
| `title` | String | Required |
| `genres` | List\<String\> | e.g. `["Sci-Fi", "Thriller"]` |
| `releaseYear` | Integer | Min 1990, max current year + 10 |
| `description` | String | Max 2000 characters |
| `posterUrl` | String | |
| `averageRating` | Double | Set by rating-service; null until first rating |
| `createdAt` | LocalDateTime | Set automatically on save |

## Common Error Responses

All errors follow [RFC 9457 Problem Details](https://www.rfc-editor.org/rfc/rfc9457):

```json
{
  "type": "about:blank",
  "title": "Bad Request",
  "status": 400,
  "detail": "Movie not found with id: 3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "instance": "/api/movies/3fa85f64-5717-4562-b3fc-2c963f66afa6"
}
```

Validation errors include a field-level `errors` map:

```json
{
  "type": "about:blank",
  "title": "Validation Error",
  "status": 400,
  "detail": "Validation failed",
  "errors": {
    "title": "Title is required"
  }
}
```

| Status | Meaning |
|--------|---------|
| 400 | Validation failure or resource not found |
| 401 | Missing or invalid JWT |
| 403 | Insufficient role |
| 422 | Upstream service unavailable (e.g. movie-service unreachable from user-microservice) |
| 500 | Unexpected server error |
